# Glossary

## Terms

- **Install channel:** The installer family, edition, and scope that owns the active command and must be preserved by `sd300 update`.
- **Latest intent:** A deliberately launched fresh official install may replace the previous owner, including with the same or an older version.
- **Observation:** A diagnostic value plus availability, validity, freshness, source, scope, unit, confidence, access tier, and sensitivity.

## Acronyms

- **MSI:** Windows Installer package.
- **PKG:** macOS Installer package.
- **WMI:** Windows Management Instrumentation.
- **PDH:** Windows Performance Data Helper.
